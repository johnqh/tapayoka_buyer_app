# tapayoka_buyer_app

Landing page web app for Tapayoka. Directs users to download the native buyer app.

## Setup

```bash
bun install
bun run dev          # http://localhost:5211
```

## Stack

Vite, React 19, TypeScript, Tailwind CSS.

## Development

```bash
bun run dev          # Vite dev server (port 5211)
bun run build        # Production build
bun run typecheck    # TypeScript check
bun run lint         # ESLint
```

## Related Packages

- `tapayoka_buyer_app_rn` -- Native buyer app (iOS/Android)
- `tapayoka_api` -- Backend API server
- `tapayoka_vendor_app` -- Vendor web dashboard
